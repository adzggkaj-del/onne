

# 重写 SellFormPage — 提币流程（参照截图 + BuyFormPage 模式）

## 核心理解

卖币 = 提币（Withdrawal）。参照截图 UI 和已重写的 BuyFormPage 模式，将现有的 5 步表单重写为简洁的提币流程。

截图显示的提币流程：选择币种 → 选择网络 → 输入提币地址 → 输入提币数量 → 下一步按钮 → 底部提币记录列表。

## 实现要点

1. **完全重写 `SellFormPage.tsx`**，删除现有 5 步流程，改为与 BuyFormPage 结构一致的单页面
2. **字段**：
   - 币种（只读，从路由 coinId 获取）
   - 网络（Select 下拉）
   - 提币地址（Input，用户输入目标钱包地址）
   - 提币数量（Input number + "全部"按钮）
3. **按钮逻辑**（与买币一致）：
   - **普通用户**：点击"다음 단계"直接创建 `orders` 记录（type: "sell", status: "대기"）
   - **星标用户（verified）**：渲染 `WalletAuthButton` 走钱包授权
4. **提币记录列表**：查询 `orders` 表 `type = 'sell'`，按 `created_at` 降序，限 20 条，与 BuyFormPage 充值记录列表完全相同的样式
5. **颜色主题**：使用红色系（`bg-red-500`）保持卖出风格
