

# 重写借贷表单页 — 严格按截图还原

## 截图分析

截图展示的借贷页面是一个**单页滚动式**布局，从上到下依次为：

1. **"충비" (充币) 标题**
2. **选择币种** — 带勾选图标的 accordion 风格，下拉选择币种（如 USDT）
3. **网络** — 带勾选图标的 accordion 风格，下拉选择网络
4. **数量** — 预设金额按钮选择（1000/2000/3000/5000/10000）
5. **贷款周期** — 下拉选择周期（7/15/30/90/180）
6. **摘要卡片** — 紫色顶线卡片，包含：네트워크、대출 코인、대출 기간、총 이자율、대출 비율(100%)、대출 금액、총 이자、만기 상환 금액、상환 예정일
7. **钱包认证提示** — 黄色提示卡片 + "连接钱包并授权"紫色按钮 + USDT 승인 금액
8. **USDT 충비 기록** — 底部借贷历史记录列表（币种、时间、数量、状态）

## 与当前实现的差异

当前 `LendingFormPage` 使用**三步向导**（网络→设置→确认），截图要求改为**单页滚动式**，所有内容在一个页面上，无步骤切换。

关键差异：
- **数量**：当前用百分比滑块，截图用**固定金额按钮**（1000/2000/3000/5000/10000）
- **贷款周期**：当前用 lending_plans 的卡片网格，截图用**下拉选择器**
- **布局**：当前三步 wizard → 截图单页垂直滚动
- **底部记录**：当前无历史记录 → 截图有 USDT 充币记录列表

### LendingFormPage 重写内容

**UI 结构（从上到下）：**
1. 返回按钮 + "충비" 标题
2. `(✓) 선택 비종` — Select 下拉选择币种（默认选中 URL 参数的币种）
3. `(✓) 네트워크` — Select 下拉选择网络（chains）
4. `수량` — 预设金额按钮组（1000/2000/3000/5000/10000），点击选中高亮
5. `대출 주기` — Select 下拉选择周期（从 lending_plans 获取可用周期）
6. 摘要卡片 — 紫色顶线 Card，逐行展示所有计算结果
7. 钱包认证区 — 黄色提示卡片 + WalletAuthButton（紫色按钮） + USDT 预估金额
8. USDT 충비 기록 — 查询最近20条 `type="lending"` 的订单，展示为列表

**计算逻辑保持不变：**
- `loanKrw = selectedAmount`（直接用选择的金额，非百分比）
- `totalInterest = loanKrw * selectedPlan.interest_rate`
- `totalRepay = loanKrw + totalInterest`
- `usdtAmount = totalRepay / krwRate`
