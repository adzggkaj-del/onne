

# 重写资产页提币弹窗 — 严格按截图还原

## 截图分析

截图展示了一个完整的提币流程页面（在弹窗内），包含：
1. **选择币种** — 下拉选择（如 USDT）
2. **选择收款人** — 网络下拉、数量输入、地址输入
3. **输入提币数量** — 钱包授权提示卡片 + 连接钱包并授权按钮
4. **提币记录** — 底部显示历史记录（币种、时间、提币数量、状态）

## 方案

完全重写 `src/components/assets/WithdrawDialog.tsx`，从当前的银行信息表单改为截图中的加密货币提币流程：

### 核心改动：
- **币种选择**：下拉选择币种（复用 `useCryptoData`）
- **网络选择**：下拉选择链（复用 `chains`）

- **钱包授权**：verified 用户显示 `WalletAuthButton`，普通用户显示普通提交按钮
- **提币记录**：底部显示最近20条 `type="sell"` 的订单
- 弹窗尺寸改为 `max-w-lg`，支持滚动
