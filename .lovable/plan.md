

## CryptoX 加密货币交易平台开发计划

### 第一步：全局基础架构
- 配置暗色主题色彩系统（#0D0D1A 主背景、#141426 卡片背景、紫色渐变等）
- 搭建响应式布局框架：PC端左侧边栏(240px) + 手机端底部Tab Bar
- 实现顶部导航栏（Logo、搜索框、通知、登录/注册按钮）
- 配置 Inter + Noto Sans KR 字体

### 第二步：首页（HomePage）
- Hero Banner 渐变横幅（标题、按钮、装饰性地球SVG）
- 4个统计数据卡片（交易量、用户数、上架币种、响应时间）
- 我的资产组合卡片 + 实时新闻卡片（PC端双列）
- 人气币种行情列表（排名、币种、价格×1.01、24h涨跌、迷你图表、交易按钮）
- 点击行展开交易抽屉（买入/卖出Tab + 数量输入 + 韩币换算）

### 第三步：买币页（BuyPage）
- 分步表单：选择主链 → 选择币种（动态过滤）→ 购买数量 → 韩币实时换算 → 钱包地址 → 手续费明细 → 确认购买
- 表单验证与按钮状态管理

### 第四步：卖币页（SellPage）
- 分步表单：选择主链 → 选择币种 → 卖出数量 → 平台充币地址展示（带复制功能）→ 银行收款信息 → 提交卖单
- 订单确认弹窗与状态流转展示（等待充币→处理中→已打款→已完成）

### 第五步：借贷页（LendingPage）
- 选择主链和币种 → Slider+输入框选择借贷数量 → 借贷摘要卡片（金额、利率0.1%/天、期限30天、到期还款）→ 确认借贷

### 第六步：资产页（AssetsPage）
- 用户信息区（头像、用户名、UID、认证状态）
- 余额双卡片（账户余额渐变卡 + 平台赠金卡）
- 快捷操作按钮（充值、提现、转账、收款码）
- 持仓资产列表 + 订单记录（带筛选Tab）
- 空状态设计（插图+引导按钮）

### 第七步：交互与细节优化
- 页面切换动画（平滑过渡）
- 价格变化高亮（红/绿闪烁）
- 移动端底部抽屉下滑关闭手势
- 韩币千位分隔符格式化
- 所有价格统一×1.01上浮逻辑
